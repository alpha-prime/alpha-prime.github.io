要了解DNS解析过程首先要了解域名的层级结构

```
主机名.次级域名.顶级域名.根域名
# 即
host.sld.tld.root
```

DNS大致解析过程

```shell
从"根域名服务器"查到"顶级域名服务器"的NS记录和A记录（IP地址）

从"顶级域名服务器"查到"次级域名服务器"的NS记录和A记录（IP地址）

从"次级域名服务器"查出"主机名"的IP地址
```

`dig` 命令的 `+trace` 参数可以显示DNS的整个分级查询过程

```shell
dig +trace xiaomingyang.com
```
